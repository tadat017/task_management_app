# ## テーブル設計

## Tasks テーブル

| Column             | Type   | Options     |
| ------------------ | ------ | ----------- |
|title	             |string	|タスクのタイトル
|description	       |text	  |タスクの詳細
|deadline	           |datetime|	タスクの期限
|completed	         |boolean |	タスクの完了状態 (true/false)
|priority	           |integer |優先度（0=低, 1=中, 2=高）
